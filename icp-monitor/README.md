# ICP Monitor

Mini app locale per controllare:

- identita` ICP
- principal
- account ICP da finanziare
- saldo ICP
- saldo cycles
- stato dei canister

## Avvio

```powershell
cd "C:\Users\Luigi\Desktop\icp fatture\fatturavault\icp-monitor"
npm run dev
```

Poi apri:

[http://127.0.0.1:4180](http://127.0.0.1:4180)

## Configurazione

Modifica:

`config.local.json`

Per cambiare:

- `identity`
- `network`
- `fundingAccount`
- lista `canisters`
